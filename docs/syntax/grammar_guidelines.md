# 语法规范
## 缩进
TAB_SIZE = 4
**不要混用 `\t` 或 空格。虽然并不强制要求**

## 标识符
#### 格式:
|  类型   | 规范  |
|  ----  | ----  |
| 类/结构体/枚举/Trait等  | 大驼峰 |
| 常量  | 大写下划线 |
| 变量/函数  | 小写下划线 |

## 控制流
**if, while, for(`未来版本`)** 控制流语句/表达式语法规范为如下

#### 格式:
前跟**关键字** 后跟**条件(如果有)**,  
左大括号**不换行**, 条件不应**有多余的括号**(尤其是对习惯 c-style 语言的人来说)

#### 例:
```typedant
if true {
    0
} else {
    1
}
```

#### 反例:
```typedant
if (true)
{
    0
}
else
{
    1
}
```

## 空格
### 逗号
逗号**前不空格，后空一格**
#### 例:
```typedant
f(1, 2, 3)
```

#### 反例:
```typedant
f(1,2,3)
f(1, 2,3)
```

### 赋值
等号**前空一格，后空一格**
#### 例:
```typedant
a = 1;
```

#### 反例:
```typedant
a= 1;
a =1;
a=1;
```
