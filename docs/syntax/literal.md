# 字面量
## 类型

目前版本拥有以下类型的字面量:  
 - [整数字面量](#整数)   
 - [字符串字面量](#字符串)  
 - [布尔字面量](#布尔值)  

## 整数

**在当前版本中 整数字面量需在数字后带上对应类型 这在未来版本中将是可选项**

#### 格式:
```typedant
[num][u64/u32/u16/u8/usize/i64/i32/i16/i8/isize]
```
#### 例:
```typedant
114514191isize
2741328965usize
10u8
65534u16
0i32
```

## 字符串

字符串由两个引号包裹

#### 格式:
```typedant
"" // 在引号中间输入文本
```
#### 例:
```typedant
"请输入文本"
"点击输入文本"
...
```

## 布尔值

只有两个值, true 或 false
